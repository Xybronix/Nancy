# Dossier Musique

Placez ici le fichier audio de la chanson de Naza "Aujourd'hui c'est ton jour, c'est ton anniversaire".

## Instructions

1. Téléchargez ou obtenez le fichier audio (format MP3 recommandé)
2. Nommez-le `naza-anniversaire.mp3`
3. Placez-le dans ce dossier (`public/music/`)

Le fichier sera automatiquement chargé par le composant `BackgroundMusic`.

## Formats supportés

- MP3 (recommandé)
- OGG
- WAV
- M4A

## Note

Si vous utilisez une vidéo YouTube, vous devrez d'abord la convertir en fichier audio en utilisant un service en ligne.
